# Curated Asset Library

The full Framer export lives in `public/framer-assets`, but the rebuild should prefer the named files in `public/metanova-assets`.

## Folder structure

- `public/metanova-assets/brand`
  - logos, wordmarks, and app icons
- `public/metanova-assets/hero`
  - homepage-ready stills and the extracted hero video
- `public/metanova-assets/people`
  - founder portraits and planning/team imagery
- `public/metanova-assets/services`
  - structural, development, and project-management support visuals
- `public/metanova-assets/projects`
  - residential, hospitality, and commercial project imagery

## Notes

- `public/framer-assets` remains the raw archive in case we want more material later.
- `public/metanova-assets/manifest.json` is the machine-readable index for the curated set.
- Some Framer files are still template leftovers. The curated set intentionally excludes those unless they looked useful for the MetaNova rebuild.
